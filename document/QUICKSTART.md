# Quick Start Guide
## הרצה מהירה של המערכת

---

## 🚀 הרצה ב-3 צעדים

### 1. ודא שהסביבה מוכנה
```bash
# בדיקה מהירה:
python test_imports.py
```

**אם הכל OK - עבור לשלב 2**

אם יש שגיאות:
```bash
pip install -r requirements.txt
```

### 2. הרץ את האפליקציה
```bash
streamlit run app.py
```

### 3. השתמש באפליקציה
1. פתח דפדפן ב-`http://localhost:8501`
2. העלה תמונת תשבץ
3. התאם גריד (Coarse → Fine)
4. **סמן ✓ "השתמש ב-Pipeline מקומי"**
5. לחץ "הפעל זיהוי"

---

## ⚠️ אם יש בעיות

### שגיאה: "module has no attribute 'image_to_url'"
```bash
# תיקון:
pip uninstall -y streamlit-drawable-canvas-fix
pip install streamlit-drawable-canvas==0.9.3

# בדיקה:
python test_imports.py
```

### שגיאה: "No module named 'easyocr'"
```bash
# התקנה:
pip install easyocr paddleocr torch

# הערה: ההורדה הראשונה של EasyOCR ~200MB
```

### שגיאה: "CUDA not available"
זה בסדר! המערכת תעבוד עם CPU (קצת יותר איטית).

אם יש GPU:
```bash
# ודא שהגדרת GPU_ENABLED=True בקונפיג
```

## 📊 מה לצפות?

### ביצועים צפויים (Phase 1):
- **תשבץ 5×5**: ~3-5 שניות
- **תשבץ 10×10**: ~8-12 שניות
- **תשבץ 13×13**: ~15-20 שניות

### דיוק צפוי:
- **OCR**: 85-90% (עברית מודפסת)
- **חצים**: 90-95%
- **Confidence**: 70%+ HIGH

## 💡 טיפים

### לדיוק מקסימלי:
1. **תמונה איכותית**: לפחות 1000×1000px
2. **תאורה טובה**: ללא צללים
3. **ניגודיות**: טקסט שחור על רקע צהוב/לבן
4. **זווית**: ישר מלמעלה (לא בזווית)

### לביצועים מקסימליים:
1. סגור תוכניות אחרות
2. אם יש GPU - הפעל
3. עבוד על תמונות עד 2000×2000px

### לתצוגה טובה:
1. זום בדפדפן (Ctrl +)
2. השתמש במסך רחב
3. סדר עמודות לפי חשיבות
